CMAPI Channel Support

Meridian strives to be compliant with Common Map API as much as possible, however it deviates from this specification in instances that are suited to support Meridian requirements and user needs. The below table outlines Meridians current compliance with version 1.3.0 of CMAPI, as well outlines key differences.

API Type Channel Type CMAPI Channel MMAPI Channel Usage Notes
Core Map View Channel map.view.zoom not supported depricated N/A
Core Map View Channel not supported map.view.zoom.in extension N/A
Core Map View Channel not supported map.view.zoom.out extension N/A
Core Map View Channel not supported map.view.zoom.out extension N/A
Core Map View Channel map.view.center.overlay map.view.center.overlay standard does not support auto zoom property in payload. all calls to this method automatically zoom to bets fit for overlay
Core Map View Channel map.view.center.feature map.view.center.feature standard N/A
Core Map View Channel map.view.center.location map.view.center.location standard does not support auto zoom property in payload. all calls to this method automatically zoom to bets fit for overlay
Core Map View Channel not supported map.view.center.data extension N/A
Core Map View Channel map.view.clicked map.view.clicked extension does not support attributes for click type
