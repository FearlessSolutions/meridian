Embedding Meridian

Embedding Meridian into you application is as simple as:

1.)Creating a HTML file and inserting an iframe into it

index.html 
              <!DOCTYPE html>
<html>
  <head>
    <title>Embedded Meridian<title>
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"><script>
    <script src="js/sampleEmbed.js"><script>
  <head>
  <body>
    <div id="youMapDiv">
      <iframe id="meridianMap" src="https://localhost:3000" height="720px" width="100%"><iframe>
    <div>
  <body>
<html>

1.)Constructing a message and payload to send to that iframe (leveraging HTML5 post messaging)

index.html 
              var payload = {
    "name":"Example Points Plotting",
    "overlayId":"testOverlayId1",
    "format":"geojson",
    "feature": {
        "type":"FeatureCollection",
        "features":[
        {
            "type": "Feature",
                "geometry": {
                    "type": "Point",
                    "coordinates": [0.0, 10.0]
                },
                "properties": {
                    "featureId": "f1"
                },
                "style": {
                    "height": 24,
                    "width": 24,
                    "icon": "https://cdn1.iconfinder.com/data/icons/Map-Markers-Icons-Demo-PNG/256/Map-Marker-Marker-Outside-Chartreuse.png"
                }
        }]
    },
    "zoom":false,
    "readOnly":false
}

sendMessage("map.feature.plot", payload);

Want to learn more?

A full example of embedding Meridian (including source files) can be found here

To run the example simply open the the sampleEmbed.html in either Firefox or Chrome. Meridian will load, plot 4 features, and pan/zoom to where those features are located geographically.

Additionally, Meridian utilizes the Common Map API as teh specification for how channels and their corresponding payloads are sent to the map. You can learn more this specification at www.cmapi.org

Note: The current version of CMAPI supported by Meridian is version 1.2.0. Our intent is to support as much of the specificatoin as possible and extend/override channel functionality where required. Additionally, our team continues to work with the technical community to update and refine the speification. However, if there are channels you would like to see modifiied or added please joing their Google Group here
