Embedding Meridian

Embedding Meridian into you application is as simple as:

1.)Inserting the following code

index.html 
              <!DOCTYPE html>
<html>
  <head>
    <title>Embedded Meridian<title>
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"><script>
  <head>
  <body>
    <div>
      <iframe id="webapp" src="https://localhost:3000" height="700px" width="100%"><iframe>
    <div>
  <body>
<html>
